Property¶

class Property¶

This class holds the data used in properties by the Atom and the Frame classes. A property can have various types: bool, double, string or Vector3D.

Public Types

enum Kind¶

Possible types holded by a property.

Values:

BOOL = 0¶
DOUBLE = 1¶
STRING = 2¶
VECTOR3D = 3¶
